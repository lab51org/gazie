Se hai configurato bene il tuo server web, tra 15 secondi sarai reindirizzato alla pagina relativa allo script di istallazione/aggiornamento (install.php) che si trova sulla directory /setup, per eseguire l'installazione di GAzie e del suo database oppure l'aggiornamento del solo database. Se ciò non dovesse avvenire segui il link:
INSTALLAZIONE O AGGIORNAMENTO DI GAZIE
Per poter utilizzare GAzie occorre installarlo ed eseguirlo tramite protocollo http://
Al momento invece stai navigando tramite protocollo file://
Se stai lavorando sulla macchina server ed hai copiato questi file sulla sua directory radice
potrebbe essere sufficiente cliccare questo indirizzo:

"http://localhost/gazie/setup/install/install.php"

se hai copiato GAzie su un alias prova a modificare "localhost" con il giusto indirizzo.

Se non hai successo segui le indicazioni sul file
INSTALL.html
presente nella directory /doc
